GITHUB

cos'e' github?
Github è un sito online dove milioni e milioni di programmatori di tutto il mondo possono caricare il proprio progetto, quindi file cartelle e tutto ciò che è necessario per realizzare un sito o web o un qualsiasi programma di qualsiasi linguaggio. I progetti che carichiamo è possibile renderli pubblici oppure privati, è possibile vedere uno storico dei file che abbiamo caricato, quindi possiamo vedere com'era un progetto nostro qualche giorno fa e nel caso ripristinarlo. Github è molto utile anche per i lavori a gruppo, è possibile caricare un progetto condividerlo con altri menbri che posso lavoraci assieme a noi. Per prima cosa nel sito di github si possono inserire delle repository, ovvero dei contenitori con tutti i file del nostro progetto. Per caricare i nostri progetti su github è indispensabile installare sul nostro pc githubdesktop. Tramite githubdsktop posso clonare i lavori e quindi trasferire una copia o da pc a sito github o da github a pc . Una volta clonato il file sul mio pc io posso lavorarci e creare un commit ,quindi un cambiamento del file originale, possiamo trasferire più commit una volta terminati i cambiamenti posso trasferire il mio nuovo lavoro sulla repository attraverso un cosiddetto "push" , un altra funzione importante di github sono i branch ovvero creare un nuovo ramo del progetto creando delle modifiche ma potendo preservare il ramo originale,quando andrò ad unire i due rami è perchè sono certo del codice che ho creato e quindi vado ad unificare i due rami attraverso una "pull request" , dando un nome e una descrizione la fusione può essere eseguita automaticamente da github (able to marge) in altri è necessario unirli a mano.. Come hostare il nostro codice su github
La prima cosa da fare per hostare un sito attraverso github è chiaramente creare una repository, quindi entro nel mio profilo di github , vado sulle repository , premo il tasto new , ne do il nome , come opzione scelgo pubblica altrimenti non posso ostare , e clicco su create repository . bene ora vado su githubdesktop
