1. Cos'è il CSS?
CSS (Cascading Style Sheets) serve a dare stile a una pagina HTML. Senza CSS, una pagina HTML sarebbe una "tristissima pagina bianca" con un font uniforme, senza colori o altre personalizzazioni. CSS permette di trasformare una pagina HTML base in qualcosa di più accattivante, aggiungendo colori, modificando font, e disponendo elementi come bottoni.

2. Come inserire CSS in HTML
Esistono tre modi principali per inserire CSS in un documento HTML:
• File esterno: È il metodo più comune e consigliato per progetti di una certa dimensione. Si crea un file .css separato (es. style.css) e lo si collega all'HTML usando il tag <link rel="stylesheet" href="style.css">. Questo permette di riutilizzare lo stesso stile su più pagine, rendendo la gestione e le modifiche molto più efficienti.
• CSS interno: Si scrivono le regole CSS direttamente all'interno del tag <style> nella sezione <head> del documento HTML.
• CSS in linea (inline): Si applicano le regole CSS direttamente a un singolo elemento HTML utilizzando l'attributo style. Ad esempio: <p style="color: red;">Testo rosso</p>.

3. Come funziona nella pratica
Quando si carica una pagina web, il browser segue questi passaggi:
1. Viene caricato e letto l'HTML.
2. Viene caricato e letto il CSS.
3. Viene creato il DOM (Document Object Model).
4. Il CSS viene applicato agli elementi del DOM.
5. La pagina viene mostrata all'utente con lo stile applicato.

4. Il DOM (Document Object Model)
Il DOM è una rappresentazione ad oggetti del documento HTML. Trasforma gli elementi HTML (come <html>, <body>, <h1>, <p>) in una struttura ad albero di nodi.
• Parent/Child: <html> è padre di <head> e <body>.
• Sibling: <head> e <body> sono fratelli.

5. Prime regole CSS
Una regola CSS base viene applicata a un selettore e definisce una o più proprietà con valori.
Esempio:
h1 {
  background-color: aqua;
  border: solid 1px black;
  padding: 20px;
}
Se una proprietà è scritta male (es. font color invece di color), il browser la ignora senza errori.

6. CSS e browser
Ogni browser supporta le proprietà CSS in modo diverso. Per controllare la compatibilità si usa caniuse.com.

7. La Sintassi
Una regola CSS è composta da:
• Selettore
• Parentesi graffe { }
• Dichiarazioni proprietà: valore;

8. Selettore e regole
Tipi principali:
• Selettore di elemento: h1
• Selettore di classe: .miaClasse
• Selettore di ID: #mioId

9. Proprietà e Valore
Proprietà = cosa cambiare (es. background-color)
Valore = come cambiarlo (es. aqua)

10. Cascata
Se due regole si applicano alla stessa proprietà, vince l’ultima scritta. Es:
h1 { color: blue; }
h1 { color: red; }
Risultato: il testo sarà rosso.

11. Specificità
Ogni selettore ha un punteggio:
• Elemento: 1
• Classe: 10
• ID: 100
• Inline style: 1000
Regole con punteggio maggiore vincono. !important forza la priorità, ma va usato poco.

12. Ereditarietà
Alcune proprietà si ereditano dai genitori (es. color).
• inherit → eredita
• initial → valore di default
• unset → comportamento naturale

13. Resettare i valori
Con all: unset; si resettano tutte le proprietà di un elemento.
